Steps to run the applications

1) After extracting, open the Server folder.
2) Launch My2DAttempt.exe.
3) Then open the Client folder.
4) Launch My2DAttempt.exe.
5) Enter the IP displayed on the Server in the Client Application and click Enter.
6) Use W,S,A,D or arrow keys to control the player and use spacebar to Jump.
